## 常用购票人页面

>本文档用于常用购票人页面接口

### 1. 根据当前用户返回当前用户常用购票人列表

>本接口用于常用购票人列表查询

#### INPUT

>无

#### OUTPUT

>data属性(数组)的数据格式为

<table>
    <tr>
        <td>字段</td>
        <td>字段类型</td>
        <td>字段说明</td>
    </tr>
    <tr>
        <td>linkId</td>
        <td>整形</td>
        <td>联系人Id</td>
    </tr>
    <tr>
        <td>name</td>
        <td>字符串</td>
        <td>用户姓名</td>
    </tr>
    <tr>
        <td>idCard</td>
        <td>字符串</td>
        <td>身份证号码</td>
    </tr>
    <tr>
        <td>cardType</td>
        <td>字符串</td>
        <td>卡类型(0:身份证,1:军官证)</td>
    </tr>
</table>
